## 데모 실행 방법

### 1. 설치

```bash
git clone https://github.com/dusttin/academy-parent-chatbot-demo
cd academy-parent-chatbot-demo
pip install -r requirements.txt
```

### 2. 환경 변수 설정

```bash
cp .env.example .env
# .env 파일에 OpenAI API 키 입력
```

### 3. 실행

```bash
uvicorn main:app --reload
```

브라우저에서 `http://localhost:8000` 접속 →
**"샘플로 바로 체험"** 버튼 클릭 → 학부모 입장에서 문의 입력

---

## 동작 방식

```
학원 안내문 PDF 업로드
       ↓
ChromaDB 벡터 인덱싱
       ↓
학부모 문의 입력
       ↓
유사 내용 검색 (RAG)
       ↓
gpt-4o-mini 답변 생성
       ↓
학부모에게 즉시 응답
```
